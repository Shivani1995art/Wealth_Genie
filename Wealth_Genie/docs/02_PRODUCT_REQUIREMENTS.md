# Product Requirements

## Supported Document Types (MVP)

| Type | Notes |
|------|-------|
| Bank Statements | PDF — savings, current, and salary accounts |
| Credit Card Statements | PDF |
| Loan Documents | PDF — home, personal, auto, education |
| Salary Slips | PDF |

All documents must be uploaded as PDF. Other formats (Excel, images, scanned
non-searchable PDFs) are out of scope for MVP.
